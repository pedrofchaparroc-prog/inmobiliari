# Hola Flask

Proyecto Flask básico que muestra "¡Hola, mundo desde Flask!"

Instrucciones para Windows PowerShell:

1. Abrir PowerShell y situarse en la carpeta del proyecto:

   ```powershell
   cd "c:\Users\LENOVO PC\OneDrive\Documentos\inmobiliari\hola_flask"
   ```

2. Crear y activar un entorno virtual (opcional pero recomendado):

   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   ```

3. Instalar dependencias:

   ```powershell
   pip install -r requirements.txt
   ```

4. Ejecutar la aplicación:

   ```powershell
   python run.py
   ```

5. Abrir en el navegador: `http://127.0.0.1:5000` o `http://localhost:5000`.
